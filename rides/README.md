# Анализ сервиса аренды самокатов

Проект посвящён анализу работы сервиса аренды самокатов GoFast на основе данных о пользователях, поездках и подписках.  
Цель — понять поведение клиентов, проверить гипотезы продакт‑менеджеров и помочь маркетингу и техподдержке в принятии решений.

## Цели исследования

- Изучить данные о пользователях, поездках и тарифах.  
- Обработать пропуски, ошибки и дубликаты.  
- Провести исследовательский анализ: активность пользователей, длительность и дистанция поездок, выручка.  
- Построить агрегаты помесячной выручки и активности.  
- Проверить статистические гипотезы для продакт‑менеджеров.  
- Оценить сценарии для отдела маркетинга (промокоды, push‑уведомления).  
- Сформулировать выводы и рекомендации для бизнеса.

---

## Данные

Используются три основных датасета:

### users_go.csv — пользователи
- `user_id` — идентификатор пользователя  
- `age` — возраст  
- `city` — город  
- `subscription_type` — тип подписки (`free`, `ultra`)  

### rides_go.csv — поездки
- `user_id` — идентификатор пользователя  
- `distance` — дистанция поездки (метры)  
- `duration` — длительность поездки (минуты)  
- `date` — дата поездки  

### subscriptions_go.csv — подписки
- `subscription_type` — тип подписки  
- `minute_price` — стоимость минуты поездки  
- `start_ride_price` — стоимость старта  
- `subscription_fee` — ежемесячная стоимость подписки

## Общий вывод

В ходе работы:

- Загружены и исследованы данные о пользователях, поездках и подписках.  
- Проведена предобработка: исправлены типы, удалены дубликаты, созданы новые признаки.  
- Выполнен исследовательский анализ поведения пользователей и выручки.  
- Построен агрегированный датафрейм с помесячной выручкой.  
- Проверены ключевые гипотезы:
  - подписчики катаются дольше;  
  - не превышают оптимальную дистанцию поездки;  
  - приносят больше выручки.  
- Даны рекомендации для маркетинга по промокодам и push‑уведомлениям.

Главный вывод: **пользователи с подпиской более выгодны для компании**, так как приносят больше выручки и демонстрируют более высокую вовлечённость, при этом нагрузка на самокаты остаётся в допустимых пределах.
