# Определение возраста по фотографии

## Цель проекта

Построить модель компьютерного зрения, которая по фотографии лица
определяет возраст человека.\
Модель обучена на датасете **ChaLearn Looking at People** и достигает
качества **MAE ≈ 6.27**, что удовлетворяет требованиям (\< 7).

## Данные

Используется датасет:

    datasets/faces/
     ├── labels.csv
     └── final_files/

**labels.csv** содержит два столбца:

-   `file_name`
-   `real_age`

Всего изображений: **7591**

## Исследовательский анализ данных (EDA)

### Основные наблюдения

-   Средний возраст: **31 год**
-   Больше всего людей 20--40 лет
-   Дисбаланс по детям и пожилым
-   Большая вариативность качества снимков

## Подготовка данных

Используется `ImageDataGenerator`:

-   `rescale=1/255`
-   `validation_split=0.25`
-   `flow_from_dataframe`

Функции: - `load_train(path)` - `load_test(path)`

## Архитектура модели

Backbone: **ResNet50** (без верхних слоёв)

Сверху: - GlobalAveragePooling2D - Dense(256, relu) - Dense(64, relu) -
Dense(1, linear)

Параметры: - Loss: MSE
- Metric: MAE
- Optimizer: Adam (lr=1e-4)

## Обучение модели

`train_model(model, train_flow, test_flow, epochs=10)`

Результаты: - Train MAE: 10.27 → 2.70 - Val MAE: падает и
стабилизируется на \~6.2

## Итоговые метрики

    Test MAE: 6.2721

## Анализ результата

-   Модель хорошо предсказывает частые возраста (20--40)
-   Ошибается чаще на редких (дети, пожилые)
-   Оптимальное число эпох ≈ 8
