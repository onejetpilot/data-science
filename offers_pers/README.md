# Персонализация предложений

Проект посвящён разработке модели, которая определяет наиболее выгодные персональные предложения для клиентов интернет‑магазина «В один клик». Цель — повысить покупательскую активность и увеличить прибыль за счёт индивидуальных рекомендаций.

## Цели исследования

- Изучить данные о поведении пользователей на сайте, их покупках и прибыли.  
- Провести очистку, объединение и предобработку данных из нескольких таблиц.  
- Выполнить исследовательский анализ (EDA) для выявления закономерностей.  
- Разработать модель сегментации / персонализации.  
- Оценить влияние персонализации на прибыль.  
- Сформулировать рекомендации для внедрения в бизнес.

---

## Данные

В проекте используются несколько таблиц:

### **market_file.csv**
Поведенческие данные пользователей за последние 3 месяца:
- количество визитов,
- количество просмотренных товаров,
- среднее число просмотров за визит,
- количество добавлений в корзину и др.

### **market_money.csv**
Выручка по пользователям в разрезе:
- период («текущий месяц» / «предыдущий месяц»),
- сумма выручки.

### **market_time.csv**
Время, проведённое пользователем на сайте (в минутах).

### **money.csv**
Данные о **среднемесячной прибыли** каждого покупателя внутри корпоративной базы.

Все таблицы объединяются по ключу **user_id**.

## Результаты

### Основные выводы модели:
- наиболее предсказуемо ведут себя пользователи с высокой активностью;  
- модель даёт уверенное разделение между «купит / не купит»;  
- персонализация позволяет увеличить прибыль за счёт:
  - приоритетной отправки предложений клиентам с высокой вероятностью отклика,
  - исключения низкоэффективных групп, чтобы снизить расходы.

### Экономический эффект:
- максимальный рост прибыли достигается при выборе сегментов с высокой вероятностью отклика;  
- убыточные сегменты (низкая прибыль + низкая активность) нецелесообразно стимулировать.

---

## Основные выводы

- Данные успешно объединены и очищены.  
- Проведён детальный EDA, определены ключевые поведенческие признаки.  
- Построена модель персонализации, определяющая вероятность отклика на предложение.  
- Персонализация позволяет существенно увеличить прибыль.  
- Оптимальная стратегия: фокус на активных и среднеактивных пользователях с высокой прибылью.
